blancoDbCommon は R/Oマッピングツール blancoDbシリーズの共通部分を提供するものです。
比較的 具体的な共通処理について、このプロジェクトで担います。

blancoDbについては、下記のURLで入手することができます。
●http://hp.vector.co.jp/authors/VA027994/blanco/blancodb.html
